# CoSteward CoScendence Rubric (v1)

Seeded: 20251206T170639Z

This file extracts the rubric from the practice guide for easier reuse in tools and dashboards.

## Rubric Table

| Pillar                        | Description                                      |
|------------------------------|--------------------------------------------------|
| Care for fragile beings      | How readily you notice and protect the vulnerable. |
| Willingness to repair harm   | How quickly and sincerely you move to repair.   |
| Active love (goodwill + duty)| How often goodwill turns into concrete support. |
| Hope under pressure          | How much you can still act toward better futures when under strain. |
| Humility under praise        | How little praise distorts your judgement.      |
| Clarity under conflict       | How clearly and fairly you see others in conflict. |
| Stewardship over ownership   | How often you choose responsibility over control. |
| Comfort with transparency    | How willing you are to let others see how decisions are made. |
| Ability to refuse harmful power | How ready you are to walk away from harmful advantages. |
| Ability to invite critique   | How often you actively ask others to question you. |

Intended use:
- personal journaling,
- CoSteward dashboards,
- gentle check ins, never hard gates.
