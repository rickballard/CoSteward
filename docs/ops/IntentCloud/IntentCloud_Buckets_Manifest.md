# IntentCloud Buckets Manifest (seed-stage v0)

Status: EXPERIMENTAL / SEED
Mission id: CoPrime_IntentCloud2025_Mission
Line: Co1 (UNBOUNDED)

This manifest maps **high-level buckets** of work to the IntentCloud surfaces
created in MW1–MW2. It does *not* list every intent; it just shows where to look first.

Related docs:
- Inventory: docs/ops/IntentCloud_Inventory.md
- Surfaces index: docs/ops/IntentCloud/IntentCloud_SurfacesIndex.md

Buckets are mission-local heuristics for CoPrime_IntentCloud2025_Mission.
Future missions can refine or extend this mapping.

## 1. Buckets overview

- CoTheory: CoCivium theory / CoTheory-of-everything / deep frames.
- CoEvo: evolution rails, CoEvo theory, auto-evolution helpers.
- CognoCarta: CognoCarta / Consenti / civic-contract frames.
- CoCore: schemas, best-practice domains, structured models.
- Orchestration: CoAgent / CoArena / CoSyncBus / megascroller helpers.
- Websites: narrative surfaces and public-facing assets.
- Archives: legacy material and CoCivium archive rehydration hints.
- MetaRails: CoAudit, CoStatus, BPOEs, IntentCloud itself.
