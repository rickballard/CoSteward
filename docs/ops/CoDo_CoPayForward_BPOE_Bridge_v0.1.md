## 1. BPOE expectations

BPOE assumes:

- Serious sessions are **CoDo-aligned by default**.
- CoStewards and Co1 instances:
  - ask for clear intents,
  - prefer repo-first artifacts,
  - leave behind **giftable outputs** per Wave.

## 2. Practical rules

When operating in BPOE:

- For every significant session:
  - Name the mission and Wave.
  - Expect at least one CoPayForward artifact.
  - Store it in a predictable path with a clear name.
- When a session ends with only chatter:
  - treat it as incomplete,
  - or note that a follow-up CoDo cleanup Wave is needed.
