# CoEmergence Harness Tracks (v1)

- 01_ProbeEvo: Prompt and scenario evolution (what we ask).
- 02_HarnessEvo: Metrics, scoring rubrics, and tagging evolution (how we measure).
- 03_ProtocolEvo: Loop and window structure evolution (how we sequence tests).
- 04_PopulationEvo: Model and agent mix evolution (who we test).
- 05_MetaTheoryEvo: Conceptual evolution of our emergence theory (what we think it means).

Each track folder may contain:
- README.md (track intent)
- VERSION_NOTES.md (changes over time)
- assets/ (prompt sets, rubrics, schemas, etc.)
